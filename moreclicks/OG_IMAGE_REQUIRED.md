# OG Image Required

## ⚠️ Action Required

You need to create an Open Graph image for social media previews.

### File Location
Create: `/public/og-image.png`

### Specifications
- **Dimensions**: 1200×630 pixels
- **Format**: PNG (or JPG)
- **Content**: Should include:
  - MoreClicks logo
  - Brand name "MoreClicks"
  - Tagline: "AI SEO Analyzer & SEO Audit Tool"
  - Blue background (#2563EB) to match brand

### Design Suggestions
1. Use the logo from `/public/logo.svg` (scaled appropriately)
2. Add text: "MoreClicks – AI SEO Analyzer & SEO Audit Tool"
3. Use brand colors: Blue (#2563EB) background
4. Keep it clean and professional
5. Ensure text is readable at small sizes (social previews)

### Tools to Create
- Canva (1200×630 template)
- Figma
- Photoshop
- Online OG image generators

### Once Created
The image will automatically be used in:
- OpenGraph metadata (Facebook, LinkedIn)
- Twitter cards
- Social media link previews

The metadata is already configured in `src/app/layout.tsx` to use `/og-image.png`.
